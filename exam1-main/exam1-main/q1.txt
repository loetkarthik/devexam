Q1: Perform Git operations to commit, push, merge branches, and create a pull request on GitHub.

1. Initialize Repository
git init

2. Add and Commit Files
git add .
git commit -m "Initial commit"

3. Connect to Remote Repository
git remote add origin https://github.com/ViknanS/exam1.git
git branch -M main
git push -u origin main

4. Create New Branch
git checkout -b feature-branch

5. Make Changes and Commit
echo "New Feature" > feature.txt
git add feature.txt
git commit -m "Added new feature"

6. Push Feature Branch
git push origin feature-branch

7. Merge Branch into Main
git checkout main
git merge feature-branch

8. Push Merged Code
git push origin main

9. Create Pull Request (GitHub Steps)
- Go to repository on GitHub
- Click "Compare & pull request"
- Add title and description
- Click "Create pull request"
- Click "Merge pull request"

Execution Summary:
git init → initialize repository
git add → stage files
git commit → save changes
git push → upload to GitHub
git branch → create branch
git merge → combine branches
pull request → merge using GitHub UI
